## 📋 Overview

<a href="#magnet">Magnet</a> serves Mac users who work with multiple windows simultaneously — developers with code editor and terminal side by side, writers with research and document windows arranged, designers comparing reference and working files, and anyone who has manually dragged and resized Mac windows more times than they can count.

The <a href="#magnet">developer workflow</a> benefits most directly from Magnet — code editor on the left two-thirds, terminal on the right third, browser preview on the second monitor, documentation in the remaining quarter. This complex workspace that would require minutes of manual window positioning configures in seconds with keyboard shortcuts. The <a href="#magnet">writer and researcher workflow</a> places the writing application beside sources, references, and notes — a side-by-side reading and writing arrangement that manual window management makes tedious and Magnet makes instant.

The <a href="#magnet">multiple monitor productivity</a> uses Magnet to organize windows across extended displays — primary monitor for active work, secondary monitor for reference, communication, and monitoring — with windows snapped to consistent positions on each display. <a href="#magnet">One-time Mac App Store purchase</a> — Magnet is a one-time purchase from the Mac App Store with no subscription.</p>

<a href="#magnet">Almost every snap position imaginable</a> is covered — left half, right half, top, bottom, all four corners, thirds, two-thirds, center, full screen, and combinations — the complete vocabulary of window arrangement that productive Mac use requires without any position feeling like an afterthought. <a href="#magnet">Compatibility with macOS Stage Manager</a> and other macOS window features means Magnet coexists with Apple's own window management additions rather than conflicting with them.

<a href="#magnet">Lightweight background operation</a> — Magnet runs as a menu bar application with minimal resource usage, activating only when keyboard shortcuts are pressed or windows are dragged, without visible impact on Mac performance. <a href="#magnet">Long Mac App Store presence</a> — Magnet has maintained its position as the top-ranked paid productivity app in the Mac App Store for years, accumulating hundreds of thousands of ratings and establishing itself as the de-facto standard Mac window manager.</p>

## 🙋 FAQ

<details>
<summary>Can I restore a window to its original position after snapping?</summary>

Yes. A restore previous position function returns the window to its pre-snap size and position.

</details>

<details>
<summary>Does Magnet show a menu bar icon?</summary>

Yes. The menu bar icon provides a visual snap position grid as an alternative to keyboard shortcuts.

</details>

<details>
<summary>What snap positions does Magnet support?</summary>

Left half, right half, top half, bottom half, all four corners, left third, center third, right third, two-thirds combinations, center, and full screen.

</details>

<details>
<summary>Can Magnet move windows between monitors?</summary>

Yes. Keyboard shortcuts move the active window to the next or previous connected display.

</details>

<details>
<summary>What macOS version does Magnet require?</summary>

Check the Mac App Store listing for current minimum macOS requirements.

</details>
